# Split Money Application (Spring Boot)

A Spring Boot-based application that helps groups or friends easily **split expenses** and **track balances**.  
This project demonstrates clean backend architecture using **Spring Boot**, **JPA**, and **REST APIs**, ideal for learning real-world expense-sharing systems (like Splitwise).
